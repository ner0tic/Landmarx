Class LandmarkItem

Interface implemented by a landmark item.

It roughly represents a single tag and is what you should interact with most of the time by default. Originally taken from ioMenuPlugin (http://github.com/weaverryan/ioMenuPlugin)

public Landmarx\Landmark\ItemInterface
# slice( mixed $offset, mixed $length = null )

Get slice of landmark as another landmark.

Get slice of landmark as another landmark.

If offset and/or length are numeric, it works like in array_slice function:
If offset is non-negative, slice will start at the offset. If offset is negative, slice will start that far from the end.
If length is null, slice will have all elements. If length is positive, slice will have that many elements. If length is negative, slice will stop that far from the end.

It's possible to mix names/object/numeric, for example: slice("child1", 2); slice(3, $child5); Note: when using a child as limit, it will not be included in the returned landmark. the slice is done before this landmark.
